# Centro de Estética & SPA - React Application

¡Bienvenido al repositorio oficial del **Centro de Estética**! Una plataforma web moderna, elegante y totalmente funcional diseñada para ofrecer una experiencia premium a los clientes desde su primer contacto digital.

---

## Características Destacadas

* **Arquitectura de Componentes:** Código modular y reutilizable para una fácil escalabilidad.
* **Experiencia de Usuario (UX) Fluida:** * Navegación con **Glassmorphism** (efecto desenfoque) en dispositivos móviles.
* **Menú Hamburguesa** independiente para una mejor mantenibilidad.
* **Sección Exclusiva Men's Care:** Diseño diferenciado en *Dark Mode* adaptado al público masculino.
* **Detalle de Tratamientos Dinámicos:** Sistema inteligente que consume un catálogo `JSON` local y genera páginas de detalle automáticamente según la URL.
* **Animaciones de Scroll:** Integración con `Framer Motion` para una aparición elegante de los elementos al navegar.
* **Formulario de Contacto:** Diseño con iconos de `Lucide React` y validaciones.

---

## Tecnologías y Herramientas

* **React 18** (Vite/CRA)
* **React Router Dom** (Navegación y rutas dinámicas)
* **React Router Hash Link** (Anclas suaves entre secciones)
* **Framer Motion** (Animaciones y transiciones)
* **Lucide React** (Iconografía minimalista)
* **CSS3 Moderno** (Flexbox, Grid y Custom Properties)
